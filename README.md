# Mavericks Legacy

> Un proyecto experimental para transformar drásticamente la apariencia de versiones modernas de macOS en una experiencia visual nostálgica.

## 🎯 Objetivo del Proyecto

La idea principal de este repositorio es documentar, recolectar herramientas y generar las configuraciones necesarias para llevar a cabo una personalización visual extrema en macOS (actualmente apuntando a macOS Tahoe). 

Queremos tomar un sistema operativo actual, caracterizado por su diseño plano moderno, y aplicarle una cirugía estética para devolverle un estilo **totalmente retro**. Nuestra máxima inspiración es el Skeuomorfismo y la icónica estética *Aqua* de **OS X Mavericks**.

Sabemos que macOS es un ecosistema sumamente cerrado y las capas de renderizado gráfico varían versión tras versión. Por lo tanto, abordamos esto con dos enfoques:
- **Plan A:** Lograr una réplica visual lo más exacta posible a Mavericks (iconos 3D, texturas, dock reflectante como "estante" y bordes rectos).
- **Plan B:** Si las estrictas barreras de Apple lo impiden, buscar el balance para que el sistema tenga el aspecto **más retro y minimalista posible**, garantizando que sea un entorno de trabajo funcional y cómodo.

Este repositorio está vivo: las metas, los scripts y las herramientas documentadas aquí evolucionarán a medida que avancemos y probemos los límites de la personalización profunda.
